Inserting Rows in the EMPLOYEES Table
previous|next

Two new employees have been hired. Sheila has added their records to the EMPLOYEES table.

The Object Browser has many features that make Sheila's job easier:

  • It alows Sheila to view the table information and column details.
  • It provides a shortcut called "Create and Create Another" to create one record and go to the data entry screen for the next record.
  • It checks constraint violations.
  • After completing data entry, Sheila can click a Create key to view all the records in the data window.
