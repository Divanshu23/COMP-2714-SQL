Installing Required Software

The software components that are required to develop and run PHP applications are:

Oracle Database 10g Express Edition
Apache Web Server 1.x (or later)

Zend Core for Oracle
Any text editor or tool supporting the creation of PHP source code

Prerequisites

This lesson assumes the following:

Oracle Database 10g Express Edition is installed, using the default settings with listener port 1521 and HTMLDB access on port 8080.
The SHEILA user has been created and has been loaded with the HR schema tables and data.
Your Linux machine has a default Apache Web Server installed.

You have downloaded the Zend Core for Oracle software.
You have a text editor or tool to support creation of PHP code.
