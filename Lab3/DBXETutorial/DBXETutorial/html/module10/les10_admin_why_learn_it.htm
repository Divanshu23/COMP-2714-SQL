Administration and Monitoring the Database

previous|next

In this module, you learn how to perform the basic administration tasks required to ensure you get the best performance from Oracle Database XE.

The main database administration tasks are:

  • Viewing parameter settings
  • Managing data storage
  • Managing database memory
  • Managing users
  • Monitoring the database for performance issues

You will use the Administration interface of Oracle Database XE to administer the database. You can also you the SQL command interface as an alternative

You will watch Sheila perform administrative tasks on the database. Sheila will manage tablespaces, start and shut down the database, alter users, view parameter settings and and monitor the database for performance .
