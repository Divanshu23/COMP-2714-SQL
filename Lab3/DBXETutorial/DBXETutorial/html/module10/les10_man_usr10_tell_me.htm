Managing Database Users

previous|next

You have seen how Sheila used the Administration interface to create a user in the Creating a New Database User Account section. Sheila can also use this interface for other user-management tasks, such as:

  • Locking and unlocking user accounts
  • Resetting passwords
  • Granting administrative privileges
  • Assigning quotas

In addition, the Security interface under Utilities > Object Reports enables Sheila to view the roles and privileges granted to herself.
