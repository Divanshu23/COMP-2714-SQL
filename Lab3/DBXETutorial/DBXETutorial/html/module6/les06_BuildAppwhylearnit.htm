Building an Application Using Oracle Application Express (formerly HTML DB)
previous|next

In this module, you learn how to build an application by using the Application Builder component of Oracle Database 10g Express Edition. This graphical user interface is built within the Oracle Application Express (APEX) development environment. The Application Builder is one of the components in APEX that is used to assemble an HTML interface (or application) on top of database objects such as tables and procedures.

You help Sheila create a saved query that shows all the department locations. The application that you help Sheila create will drill down to show the employees in a particular department location. The application will also enable the appropriate personnel to maintain dependent information for a particular employee.
